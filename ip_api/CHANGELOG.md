# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [1.0.0] - 2026-04-15

### Added

- Initial plugin release
- `geolocate_ip` action: look up geolocation data for a single IPv4 address, IPv6 address, or domain name
- `geolocate_bulk` action: look up geolocation data for up to 100 IPs or domains in a single batch request
