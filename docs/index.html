Importing Plugins

  1. Build the .plg file: run insight-plugin export -d plugins/<name>
  2. In InsightConnect, go to Settings > Plugins > Import
  3. Upload the .plg file from the plugin directory
  4. Configure the connection using the setup guide linked above

The .plg files are generated locally (~80-200MB each) and not stored in this repository.
