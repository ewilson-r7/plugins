# ChatGPT Plugin (v1.0.0)

Interact with OpenAI ChatGPT models to assist SOC analysts with threat analysis, incident summarization, script explanation, and response recommendations.

## Actions
- **Ask ChatGPT** - Send a prompt and get a response
- **Analyze Indicator** - Analyze a security indicator (IP, domain, hash) for threat context
- **Summarize Incident** - Generate a concise summary of a security incident
- **Suggest Response Actions** - Get recommended response actions for a threat scenario
- **Explain Script** - Explain what a script does in plain language

## Connection
- **API Key** - OpenAI API key (`sk-proj-...`)
- **Model** - Default model to use (e.g., `gpt-4o`)

## Import
Upload `rapid7_chatgpt_1.0.0.tar.gz` via InsightConnect Settings > Plugins > Import.
