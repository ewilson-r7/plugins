Description

Interact with OpenAI ChatGPT models to assist SOC analysts with threat analysis, incident summarization, script explanation, and response recommendations.

This plugin connects to the OpenAI Chat Completions API and provides structured, security-focused actions designed for use in analyst workflows within InsightConnect.

Supported Models

gpt-4o gpt-4o-mini gpt-4-turbo gpt-3.5-turbo

Key Features

  • Send prompts to ChatGPT and receive AI-generated responses
  • Analyze indicators of compromise with AI-powered threat context
  • Summarize security incidents into concise executive briefings
  • Get AI-recommended response actions for security incidents
  • Explain suspicious scripts and commands with security context

Requirements

  • OpenAI API key with access to Chat Completions API
  • Sufficient API credits/quota for the selected model

Setup & Connection

To obtain an API key, visit platform.openai.com/api-keys. Create a new secret key and copy it immediately — it will not be shown again.
NameTypeRequiredDescription
API Key credential_secret_key Required OpenAI API key for authenticating to the ChatGPT API
Model string Required Default ChatGPT model to use for completions. Options: gpt-4o, gpt-4o-mini, gpt-4-turbo, gpt-3.5-turbo. Default: gpt-4o

Ask ChatGPT

Send a prompt to ChatGPT and receive a response. This is the general-purpose action for any free-form query to the model.

Input

NameTypeRequiredDescription
Prompt string Required The prompt or question to send to ChatGPT
System Message string Optional Optional system message to set the behavior and context for the AI assistant
Max Tokens integer Optional Maximum number of tokens in the response (higher values allow longer responses). Default: 2048
Temperature float Optional Controls randomness in the response (0.0 = deterministic, 2.0 = most random). Default: 0.7

Output

Returns the text response from ChatGPT along with token usage details.

{
  "response": "Common indicators of a phishing email include suspicious sender addresses, urgent language, unexpected attachments, and mismatched URLs",
  "usage": {
    "prompt_tokens": 25,
    "completion_tokens": 150,
    "total_tokens": 175
  }
}

Analyze Indicator

Submit an indicator of compromise (IP, domain, hash, or URL) to ChatGPT for threat context analysis. Returns a structured threat assessment with risk level and recommended actions.

Input

NameTypeRequiredDescription
Indicator string Required The indicator of compromise to analyze (IP address, domain, file hash, or URL)
Indicator Type string Required The type of indicator. Options: ip_address, domain, file_hash, url
Additional Context string Optional Optional additional context about where this indicator was observed

Output

Returns a threat analysis, risk assessment, and list of recommended response actions.

{
  "analysis": "The IP address 198.51.100.23 is associated with known command-and-control infrastructure. Recommended actions: block at firewall, investigate endpoints that communicated with this IP",
  "risk_assessment": "HIGH - Associated with known C2 infrastructure",
  "recommended_actions": [
    "Block IP at perimeter firewall",
    "Search SIEM for historical connections to this IP",
    "Isolate any endpoints that have communicated with this IP"
  ]
}

Summarize Incident

Generate a concise executive summary of a security incident from provided alert data or notes. Adjusts technical depth based on the target audience.

Input

NameTypeRequiredDescription
Incident Data string Required Raw incident data, alert details, or analyst notes to summarize
Severity string Optional Severity level of the incident. Options: critical, high, medium, low, informational
Audience string Optional Target audience to adjust technical depth. Options: executive, technical, both. Default: technical

Output

Returns a concise summary, timeline of events, and list of key findings.

{
  "summary": "INCIDENT SUMMARY: Malware infection detected on WORKSTATION-045 with active C2 communications and lateral movement attempts. Immediate containment recommended.",
  "timeline": "1. Malware executed on WORKSTATION-045. 2. C2 callback established to 198.51.100.23:443. 3. Lateral movement attempted to FILE-SERVER-02.",
  "key_findings": [
    "Active C2 communication to 198.51.100.23",
    "Lateral movement attempted to FILE-SERVER-02",
    "User account jsmith potentially compromised"
  ]
}

Suggest Response Actions

Get AI-recommended response and containment actions for a security incident. Recommendations are tailored to the current incident response phase (NIST framework).

Input

NameTypeRequiredDescription
Incident Description string Required Description of the security incident requiring response recommendations
Environment Context string Optional Optional context about the environment (network segmentation, critical assets, existing controls)
Response Phase string Optional Current phase of incident response. Options: identification, containment, eradication, recovery, lessons_learned. Default: containment

Output

Returns prioritized immediate actions, investigation steps, and long-term strategic recommendations.

{
  "immediate_actions": [
    "Isolate affected workstations from the network immediately",
    "Block SMB lateral movement via host-based firewall rules",
    "Disable compromised user accounts"
  ],
  "investigation_steps": [
    "Identify patient zero and initial infection vector",
    "Search for LockBit IOCs across all endpoints",
    "Review authentication logs for compromised credentials"
  ],
  "long_term_recommendations": [
    "Implement network segmentation between workstation and server VLANs",
    "Deploy EDR with ransomware-specific detection rules",
    "Conduct phishing awareness training"
  ]
}

Explain Script

Submit a suspicious script or command for AI-powered explanation with security context. Returns a breakdown of functionality, risk assessment, MITRE ATT&CK technique mapping, and extracted IOCs.

Input

NameTypeRequiredDescription
Script string Required The script content or command to analyze and explain
Script Type string Optional The type or language of the script. Options: powershell, bash, python, batch, vbscript, javascript, unknown
Context string Optional Optional context about where the script was found (e.g., email attachment, scheduled task, registry)

Output

Returns a human-readable explanation, risk level, MITRE ATT&CK techniques, and any IOCs extracted from the script.

{
  "explanation": "This PowerShell command decodes and executes a base64-encoded script that downloads and runs content from http://example.com/malware using the .NET WebClient class",
  "risk_level": "CRITICAL - Downloads and executes remote code",
  "techniques": [
    "T1059.001 - PowerShell",
    "T1027 - Obfuscated Files or Information",
    "T1105 - Ingress Tool Transfer"
  ],
  "indicators": [
    "http://example.com/malware"
  ]
}
